Backup an SDcard using compression

Published: lun. 22 avril 2024
By cbernardini

In misc.

tags: sdcard linux

In order to backup the SDCard using compression do as follows:

sudo dd bs=4M if=/dev/sda | gzip > raspberrypi4-ubuntu`date +%d%m%y`.gz

If the SDCard is big, it will take a while. Be patient!
