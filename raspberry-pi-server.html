Raspberry Pi Server

Published: lun. 25 mars 2024
By cbernardini

In misc.

In our days, having a server at home is almost a must. Very useful to keep our backups, giving access to particular applications or controlling a domotics server. All this is possible with a raspberry pi. In addition, the power consumption is very low being around ~10 euros per year.

Requirements

  • Raspberry Pi 4 B
  • SD card (32GB)
  • Ubuntu PC (you can still do it in Windows or Mac, but the instructions given are for Ubuntu 22.04 LTS)

Install Ubuntu Server

  1. Create the SD Card image:
sudo snap install rpi-imager
rpi-imager

  1. In the rpi-imager, select the options according to your raspberry pi.

  2. Device: Raspberry Pi 4

  3. Operating System: General Purpose Os / Ubuntu / Ubuntu 22.04 LTS

  4. Storage: SDCard

  5. When you click on start, there are interesting options about customization like setting the keyboard layout, username and password, enabling ssh. In my case, I set all of them.

After waiting for the image to be written, you can remove rpi-imager

sudo snap remove rpi-imager

If you have done everything right and rpi-manager did its job, your Ubuntu machine is ready to boot.
